# CareerTrail

CareerTrail is a modern Job Application Tracking System built using Spring Boot, Spring Security, Thymeleaf, Hibernate/JPA, and MySQL.

The application helps users organize their job search by tracking applications, interview schedules, application status, and career progress through analytics.

---

## Features

### User Authentication

* Secure Login & Registration
* Password-based Authentication using Spring Security
* User-specific application management

### Application Tracking

* Add new job applications
* Edit existing applications
* Delete applications
* Search applications by company name

### Status Management

Track applications through different stages:

* Applied
* Interview Scheduled
* Selected
* Rejected

### Profile Management

* Update personal information
* Current Company
* Current Role
* Experience Level
* Current Location
* Persistent profile data storage

### Analytics Dashboard

* Total Applications
* Applied Applications
* Interview Count
* Selected Count
* Rejected Count
* Pie Chart Visualization
* Selection Rate
* Rejection Rate
* Interview Rate

### User Experience

* Dark Theme
* Light Theme
* Responsive Dashboard
* Modern UI Design
